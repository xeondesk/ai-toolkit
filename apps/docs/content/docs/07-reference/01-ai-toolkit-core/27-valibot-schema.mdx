# `valibotSchema()`

`valibotSchema` is a helper function that converts a Valibot schema into a JSON schema object
that is compatible with the AI TOOLKIT.
It takes a Valibot schema as input, and returns a typed schema.

You can use it to [generate structured data](/docs/ai-toolkit-core/generating-structured-data) and
in [tools](/docs/ai-toolkit-core/tools-and-tool-calling).

## Example

```ts
import { valibotSchema } from '@ai-toolkit/valibot';
import { object, string, array } from 'valibot';

const recipeSchema = valibotSchema(
  object({
    name: string(),
    ingredients: array(
      object({
        name: string(),
        amount: string(),
      }),
    ),
    steps: array(string()),
  }),
);
```
